The project covers:
</p>

<ul>

  <li>Subject-wise analysis</li>
  <li>Pass/Fail analysis</li>
  <li>Percentage and grade calculation</li>
  <li>Ranking and sorting</li>

<h2>Tasks Solved</h2>

<h3>Basic Analysis</h3>

<ul>
  <li>Convert dataset into NumPy array</li>
  <li>Display dataset</li>
  <li>Find:
    <ul>
      <li>Shape</li>
      <li>Dimensions</li>
      <li>Size</li>
      <li>Datatype</li>
    </ul>
  </li>
</ul>

<h3>Student-wise Analysis</h3>

<ul>
  <li>Calculate total marks of each student</li>
  <li>Calculate average marks</li>
  <li>Find class topper</li>
  <li>Find lowest scorer</li>
</ul>

<h3>Subject-wise Analysis</h3>

<ul>
  <li>Find highest marks in each subject</li>
  <li>Find lowest marks in each subject</li>
  <li>Find average marks of each subject</li>
</ul>

<h3>Pass/Fail Analysis</h3>

<ul>
  <li>Find students who passed in all subjects</li>
  <li>Find students who failed in at least one subject</li>
  <li>Display only passed students</li>
</ul>

<h3>Advanced Analysis</h3>

<ul>
  <li>Calculate percentage of each student</li>
  <li>Assign grades</li>
  <li>Find subject toppers</li>
  <li>Find overall class average</li>
  <li>Count passed and failed students</li>
</ul>
